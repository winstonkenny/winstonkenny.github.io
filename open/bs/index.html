# 毕业设计指导

支持服务类别：

STM32单片机 、物联网 、NB-IOT 、MQTT 、NFC 、RFID 、WiFi 、
Java 、Golang 、Python 、HTML 、JavaScript 、APP 、Web 、Vue

历史记录

2018年3-6月指导3位学长学姐成功毕业

2019年3-6月指导15位学长学姐成功毕业

2020年3-6月指导30位同届毕业生成功毕业

个人有丰富的毕业设计制作经验，拥有日编写上万字论文的能力，专治各种软件程序运行的疑难杂症。这里没有中间商赚差价，个人就是生产商

# 1.交付

毕设费用支付主要分为定金付50%，最终论文及实物交付30%，毕设答辩结束付20%

如果你是由熟人介绍，对我充分信任且不差钱的情况下，可以直接付全款

# 2.毕业步骤

每个学校每一年的计划都不同，会有时间差

  • 1定题目
    (寒假之前定下指导教师以及大致的题目方向，在开题报告提交之前，只要指导老师同意，都可以更改题目)
  • 2开题报告
    (大四下学期开学后的1个月内，3月初开学准备，3月末4月初提交)
  • 3中期答辩
    (5月初，进行毕业设计中期检查，论文基本写完前三章)
  • 4论文查重
    (5月中下旬，40%重复率为合格，一般要求在30%重复率，想要优秀的话要到20%以下)
  • 5最终答辩
    (6月初，实物演示，老师根据你做的设计进行简单提问)
  • 6毕设文件封存
    (6月中旬)
  • 7告别
    (6月下旬)和自己的男女朋友们、情人们、老师们、兄弟们、姐妹们一起吃个饭喝个酒散个步告个别，学校就基本没什么事了
