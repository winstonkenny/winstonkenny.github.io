# Keil开发软件介绍

本课题是利用STM32单片机来实现数据处理的，在这次的开发中，程序编译环境使用的是软件Keil uVision5，如今的Keil5软件已经能够针对大部分的芯片轻松编译，并且可以通过仿真器进行仿真，随时观察一些变量的变化情况，对程序的修改有很大的便利，极大的提高了程序的修改效率，因此受到众多开发者的喜爱。该软件可以轻松编译多种语言。本次对STM32芯片进行开发，程序编译过程中使用的编程语言则是使用了C语言程序设计课程中学到的知识，通过对C语言的学习，知道了C语言程序设计的适应范围非常广，从电脑软件到日常生活中的智能设备的使用，它可用来做一些能够解决很多实际问题的应用软件程序。用C语言编写程序，过程中有了深深的体会，它的特点就是具有非常强的逻辑性，C语言是面向过程编程的一种语言，编写过程中可以减少逻辑错误，并且可以轻松的移植到其它的硬件平台。对STM32的开发可以直接操作寄存器，或者使用库函数，对于STM32来说，寄存器非常繁多，且难以记忆，我们所写程序并不涉及到底层驱动的开发，所以可以选择简单的库函数调用，极大的降低了开发难度。并且在调试中可借鉴库函数轻松找到出错原因，开发起来得心应手。
