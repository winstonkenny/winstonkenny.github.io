# BS2020

今年大四，有充足丰富的经验指导毕业设计

发布时间：2020.03.08

解释权人：Kenny

联系方式：QQ1790218202

主要服务领域：STM32单片机、物联网、网站设计、安卓应用开发、电脑上位机开发

人员 题目 角色 选择服务 设计类型 备注
张三 基于###技术的###管理系统设计 甲方 全套 软件
李四 基于###的###系统设计与实现 甲方 全套 软件+硬件
Kenny 乙方

# 1. 服务

  1. [设计]：实物设计（纯软件设计不涉及器件费用，带有硬件部分的设计设计器件费用，关于器件费用详情见备注1）
  2. [论文]：论文正文（论文查重率达到30%以下即为服务结束，乙方提供论文正文及一篇符合查重率的查重报告）
  3. [开题]：开题报告及任务书
  4. [其他]：其他服务（由于部分老师对于论文的苛刻要求：修改标点符号、修改文章布局、修改文献注释、设计简单功能等）注：不包括难度功能升级（详情见备注4、5）
  5. [全套]：包括以上1-4所有的服务

# 2.费用

  1. 论文￥1200/人次，实物设计一人￥800/人次，二人￥750/人次，三人￥700/人次，四人及以上￥650/人次
  2. 一个人全包一整套服务，包括所有毕设过程中的文档修改￥2000
  3. 不包论文，包实物设计，需要开题报告和任务书￥50；不包实物设计，单写开题报告及任务书的话￥200
  4. 安排了介绍费50每人次，自己个人的设计走组团优惠（二人￥750、三人￥700、四人及以上￥650）的情况不计算介绍费

# 3.付款

  1. 针对所有已确定服务，实行订金制，订金金额为甲方选择的服务需求总金额的50%
  2. 付订金后即刻生效所有甲方已确定的服务内容，项目正式启动，在相应服务最终交付前甲方将尾款结清，尾款交付时间的确定，双方可进行协商

# 4.备注

  1. 设计的所有器件费用由乙方提供，乙方具有最终实物所有权，如果甲方在服务结束后需要实物，需要额外付所有的器件费用（由乙方提供淘宝购买订单详情）及购买运费
  2. 单包论文情况，甲方如果需要开题报告和任务书，提供免费撰写，不需要则乙方不提供撰写
  3. 单包论文情况，论文查重率达到30%以下，有部分设计可能会存在查重率10%以下的情况，乙方一概不负责升重服务
  4. 针对所有的服务，乙方仅负责与甲方在服务初期协商确定的设计的所有功能，不负责设计的复杂功能升级
  5. 后续要求加的功能，视功能难度情况可能会有额外收费情况，具体情况双方进行再协商
  6. 服务期间，乙方所有生产的软件源码部分都归属于甲方
  7. 乙方承诺，于定金交付日算起45天期限完成，若因乙方原因未能按时完成，甲方有权终止协议，乙方提供全额退款
  8. 若甲方要求服务提前完成，需与乙方实际商谈
  9. 与乙方同校的甲方可直接去甲方居所拿实物进行答辩，不同校的甲方需要付相应的器件费，答辩结束后邮寄给乙方，乙方退还甲方支付的器件费
  10. 本文档最终解释权归乙方所有
