# BS2019

本人今年大三，于北华大学创新实验室学习两年时间，有充足的经验为学长学姐提供毕业设计指导思路

可提供服务有 STM32单片机系统设计 、Python类软件编程 、上位机软件程序编写 、Java类软件编程

技术领域包括 RFID 、LCD 、WiFi 、NFC 、Lora、NB-IOT、Pi、Server Cloud 、Flask、 Django 、MySQL 、J2EE 、APP、Website等

注意：因51单片机资源过于贫瘠、单片机设计服务已经取消了对51单片机的支持，统一使用STM32单片机进行硬件设计。

# 1.服务

  1. 开题报告
  2. 外文翻译
  3. 设计
  4. 论文

# 2.费用

单独私聊根据设计的难度定价

# 3.付款

实行订金制，订金金额为甲方选择的服务需求总金额的50%，论文实物交付30%，答辩完成后交付20%

# 4.备注

考虑到有人答辩完成后器件无用，我负责所有器件费用，答辩完成后将器件归还于我，如果答辩后自己需要器件，则另外付器件费用。
