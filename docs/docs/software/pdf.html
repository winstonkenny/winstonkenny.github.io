# PDF文件操作

本文详细介绍如何将PDF文件内容复制到Word中

# 需要软件

  • Windows 10系统自带的Microsoft Edge软件或者使用谷歌浏览器，如果没有请更新系统或者百度安装
  • PDF源文件
  • World软件

# 步骤

  • 右键PDF文件，在打开方式中选择Microsoft Edge软件，列表中没有的选择其他应用程序之后再选择

  • 打开Word文档后将文档放置在屏幕左侧
  • 将打开的PDF文件放到右侧
  • 使用鼠标选中PDF文件中的文字，右键复制或者按键盘上的Ctrl + C组合键进行复制
  • 在Word文档中的合适位置进行粘贴
