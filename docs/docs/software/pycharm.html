# PyCharm

PyCharm是JetBrains公司旗下的一款为Python量身定制的一款代码编辑器，其丰富的代码提示功能以及软件包下载功能深受广大开发者的喜爱，在Python软件编程领域，PyCharm撑起一大片天空。PyCharm之所以如此受欢迎，主要是因为此软件提供跨平台开发服务，同时支持主流的三大操作系统，源于其软件设计使用了Java的JVM技术，不但可以让开发者在全球用户最多的微软旗下的Windows操作系统上进行代码编辑，也同样支持全世界软件工作者使用的开源的Linux操作系统，同样也支持众多平面设计师所青睐的苹果公司独有的MacOS操作系统。因为其完善的跨平台性，使得全世界的软件开发者有了舒服的开发环境。PyCharm作为一个软件开发环境，具有丰富的集成性，可以支持Django进行Python网页开发，也可以使用Flask框架进行网站开发，除此之外PyCharm也同样提供了代码分析功能、Git版本控制系统、图形调试器以及Debug测试器。

# 安装

  • 首先使用Web浏览器访问https://www.jetbrains.com/pycharm/download/网址，可以将网址内容复制到浏览器的地址栏中，点击键盘上的回车按钮即可进入PyCharm下载的官方网站，在首页可以看到Download按钮选项，选择Community社区版可以免费下载。官方首页下载界面如下图所示。

  • JetBrains公司为了更容易推广其系列软件，专门开发了ToolBox软件安装工具。下载网站为https://www.jetbrains.com/toolbox-app/双击安装后在计算机桌面系统托盘中出现ToolBox图标，点击图标即可出现如下界面。选择PyCharm Community软件进行安装即可。
