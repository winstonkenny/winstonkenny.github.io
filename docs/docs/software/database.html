# 数据库方案

一个优秀的设计是需要数据支撑的，对于数据存储方案的选择，则显得尤其重要，数据存储方案即是将系统运行过程中所产生的数据进行本地化，方便在日后的系统维护时能够有迹可循。数据本地化的方案有许多种，其中有一种是在电脑磁盘中新建文件夹，在文件夹中新建文件进行记录，但是这种方式不利于大量数据查询，为了解决数据难以查询的缺点，可以使用专业的数据库来实现对结构化数据的查询、修改和删除等操作。数据库主要分为两大类，一种是SQL类数据库，一种是NOSQL数据库，SQL数据库适合存储结构化数据，对于具有结构型的数据查询有着非常好的性能，NOSQL数据库是非关系型数据库，适合非结构化的数据存储，本设计使用的数据是结构化数据，只讨论SQL类的数据库使用，SQL类数据库主要有以下六种方案。

方案1：DB2数据库

DB2数据库是在1980年诞生的一款数据库，本身具有简单性和数据不相关性的特点，在后来的升级版本中添加了在线事务处理的支持，但因为技术较为陈旧，已经渐渐淡出人们的视野。

方案2：SQL Server数据库

SQL Server数据库是微软公司设计的一款关系型数据库管理系统，其中集成了很多强大的管理分析工具，并具有极好的可拓展性，SQL Server数据库大多数应用在大型企业级数据管理领域，强大的数据库引擎为企业级数据开发提供了更加安全可靠的应用功能，在事务处理和应用方面拥有极高的性能，因为本数据库主要面向于企业，其中有很多服务是需要收费的，不适合个人开发使用。

方案3：MySQL数据库

MySQL数据库起初是由瑞典的一家公司开发研究的一种小型关系型数据库管理系统，其以精简的设计达到了优秀的性能要求，体积相比于SQL Server数据库来说具有极大优势，不需要占用庞大的磁盘空间即可达到相对理想的性能需求。除此之外，MySQL数据库是一个开放源代码的数据库管理软件，可以个人使用此数据库的所有功能，无需额外支付其他服务费用，因为这一特点，吸引了众多小型企业，在开发自己个人公司网站时，为减少生产成本而使用MySQL数据库作为公司网站的数据库。本设计因是个人开发，所以使用MySQL数据库作为服务器的数据存储软件。

方案4：Access数据库

Access数据库是微软公司为解决办公软件中大量数据存储问题而设计的一种广西型数据库，同Word、Excel等软件一同归属于Office办公软件，其中Access数据库包含有表、查询、窗体、报表、页等七大主要对象，表主要是对具有相同特性的关联数据提供意义对应关系，查询对象这是可以对数据进行增删改的操作。Access的界面相对于前几种方案来说更加友好，使用起来更加简单灵活，在桌面数据库管理软件中，Access可以说是最为优秀的，其中因为同属于Office办公软件，所以和Office的其他办公软件之间可以做到无缝连接

# MySQL数据库安装

在浏览器里打开mysql的官网http://www.mysql.com/，页面如下所示。进入页面顶部的"Downloads"，打开页面底部的“Community(GPL) Downloads”，在页面中间的位置找到我们windows上要用的下载页面“MySQL on Windows(Installer & Tools)”，选择第一项"MySQL Installer”， 页面底端找到下载入口“ Windows (x86, 32-bit), MSI Installer ”，点击Download按钮开始下载，共381.4M。MSI格式是指windows的安装程序，下载后直接双击就能进入安装向导的那种，区别于对文件进行解压的安装方式，页面告诉询问你是否登录，告诉你登录之后有哪些好处，我们不登录，点击页面底部的“ No thanks, just start my download. ”按钮进入下载页面，开始下载，等待下载完成。双击下载好的mysql安装文件“mysql-installer-community-5.7.14.0.msi”打开安装程序，打开后需要稍等一下。选择安装类型Custom，选择Custom之后左边的安装流程和右边的描述文字会改变，然后点击"Next"按钮继续，需要从安装程序提供的可安装的产品（Products）中选择mysql server。展开Available Products里的第一项“MySQL Servers”，依次展开其子结点，直到其终端结点，选中“MySQL Server 5.7.14 - X64”，点击绿色的向右箭头，将当前Product移动需要安装的列表，然后在右边展开“MySQL Server 5.7.14 - X64”项，取消“Development Components”的勾选（因为我们只需要安装mysql server），之后点击“Next”按钮进入下一步，点击“Execute”（执行）开始安装，安装过程中会显示安装的Progress（进度），等待安装完成后Status会显示Complete，mysql图标前会出现一个绿色的勾，然后点击“Next”按钮进入产品配置界面，继续点击下一步进行安装，在Root Account Password设置数据库root账号的密码，我填的是123456所以程序提醒我密码强度为弱，需要牢记这个密码，然后点击“Next”，直到安装完成。
