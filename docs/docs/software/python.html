# Python

# 安装

  • 首先使用Web浏览器访问https://www.python.org网址，可以将网址内容复制到浏览器的地址栏中，点击键盘上的回车按钮即可进入python的官方网站，在首页可以看到Downloads按钮选项，鼠标经过时会跳出最新版本的python编译器安装包下载按钮，点击名为版本号的按钮即可跳出下载链接，下载的是3.9.2版本的python编译器，针对Windows操作系统的amd64版。官方首页下载界面如下图所示。

  • 打开下载文件夹，将下载成功的Python-3.9.2-amd64.exe使用鼠标双击打开运行，此时会弹出一个运行提示框，点击运行按钮即可弹出软件安装界面，在如下图所示的界面中点击Customize installation按钮。

  • 进入Opthonal Features功能选择界面，选择Documentation、pip、tcl/tk and IDLE、Python test suite功能，点击Next按钮进入下一步。

  • 在Advanced Options界面中将下方的Customize install location下的输入框内容修改为C:\Programs\Python\Python39，此操作一定要写清楚，在之后的环境变量配置的过程中需要此路径地址，修改完成后点击Install按钮进入软件安装界面。

  • 点击进入如下图所示的安装等待界面，等待安装进度条达到100%即可完成安装，安装等待时间一般取决于电脑的运行速度和计算机存储磁盘的读写速度。

  • 等待安装完成之后，软件界面会跳出Setup was successful字样，表示Python软件编译器已经成功安装在计算机中。

# 配置环境变量

因为在电脑使用过程中，Python软件编程需要Python编译器，已经将软件安装到了C:\Programs\Python\Python39路径下，但是计算机还不知道已经安装了Python编译器，为了让计算机知道已经成功安装了Python编译器，需要为计算机配置环境变量(Environment variable)。

  • 首先打开Windows资源管理器，在右侧空白区域中点击鼠标右键，在弹出的右侧菜单栏中点击属性按钮，即可弹出环境变量配置页面。在设置页面的右侧点击高级系统设置按钮即可打开系统属性设置框。

  • 点击系统属性设置框下方的环境变量按钮，在弹出环境变量设置框的上半部分是用户变量，双击用户变量中的Path变量，在编辑环境变量设置框中新建两个值分别为C:\Programs\Python\Python39和C:\Programs\Python\Python39\Scripts依次点击已经打开的设置框的确定按钮即可将环境变量设置完成。

# 验证环境变量配置是否成功

  • 点击键盘上的Win+R键打开运行窗口，在运行窗口中输入cmd，点击确定按钮打开命令提示符Cmd程序。

  • 在打开的命令提示符程序中输入python，然后点击键盘上的回车按钮，如果出现python 不是内部或外部命令，也不是可运行的程序或批处理文件。的提示，则表明python程序未安装成功或者是环境变量未设置成功。如果cmd窗口出现Python 3.9.2 字样表示python编译器软件安装成功且环境变量设置成功。

# 配置PIP软件安装源

python编程需要软件支持包的下载，例如python的PyQt软件包需要下载才可以使用，python默认是使用pip工具进行下载安装的，但是pip的下载安装源默认是国外服务器，因网络带宽和延迟的原因下载速度无法达到理想效果，国内有组织解决了这个问题，将国外的软件源搬到了国内服务器进行管理，为了提高下载速度，可以使用国内的软件源，在使用之前需要本地设置pip软件安装源。

  • 首先打开C盘目录，打开用户文件夹并进入用户名文件夹，文件夹路径是C:\Users\<用户名>，在此用户路径下新建一个文件夹名为pip，并在pip文件夹内右键新建一个文本文档，修改文件名为pip.ini，此时需要注意的是在查看选项卡中要勾选显示文件扩展名，如下图所示。

  • 右键pip.ini文件并在右键菜单中选择编辑按钮，打开编辑pip.ini，在文件中添加如下文本。
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple/
[install]
trusted-host = pypi.tuna.tsinghua.edu.cn

  • 点击文件保存按钮将文件进行保存，关闭编辑器即可完成配置。
