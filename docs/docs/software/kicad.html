# KiCad

KiCad是开源绘制电路原理图软件

# 下载

KiCad软件官网下载页面为Windows Downloads | KiCad EDA (opens new window)

选择自己电脑平台进行下载，本教程只展示在Windows操作系统下的安装使用

可以直接点击下方国内下载链接进行下载

国内下载链接：kicad-5.1.9_1-x86_64.exe (opens new window)

# 安装

  • 双击打开下载好的exe后缀文件，进入安装界面如下所示

  • 点击下一步

  • 选择默认选择项即可，点击下一步

  • 选择好安装目录后点击安装按钮，等待安装

  • 安装成功后界面如下所示 ，不需要勾选编辑3D对象，点击完成按钮之后完成安装

# 使用

双击打开*.sch文件即可打开并编辑电路图
