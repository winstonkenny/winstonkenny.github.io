# Proteus

Proteus是众所周知的电路图绘制软件，此软件属于是系统设计辅助软件，与芯片设计和芯片编程辅助软件同称为EDA工具软件，Proteus软件可以对复杂的电路进行模拟仿真，在国内深受教师以及单片机研发者的喜爱。本软件可以对原理图进行绘制，自动生成PCB文件并能够支持自动布线或者是人工布线。此软件是唯一一个能够将电路仿真、虚拟模型、PCB设计集为一体的EDA软件，其中最具特色的是此软件可以模拟几类单片机，通过对虚拟单片机的编程，可以在设备生产前调试单片机的软件代码，降低开发成本，在没有实际单片机开发板的条件下也能够进行软件编程。其中支持的单片机有8051、ARM、8086、MSP430等主流高校教学单片机。

软件中的器件也是极为丰富，包括马达、LED灯、LCD显示屏、AD/DA数据转换芯片、内存、键盘等实体器件，还有部分SPI器件、I2C器件。

随着科技的发展，教育资源也同步提升，在学校中，因为成本压力，无法为每一位学员都准备一套实际开发平台，虚拟开发则成为了教育领域较为流行的一种教学方式，尤其是对于一些电子设计竞赛中对电路的考核，使用仿真软件进行实训是未来教育发展的必然趋势。

# 安装

  • 双击exe扩展名的软件安装包，会出现如下Welcome to the Prerequisites Wizard安装界面，在安装界面的下方，使用鼠标点击名为下一步的按钮，进入下一步安装。

  • 在Prerequisites界面选择Visual C++ 2008 SP1软件包，此软件包是针对于Proteus软件运行时所需要的必要软件包，其他软件包不需要选中，选择完成之后点击下一步按钮进入下一步。

  • 首先会跳出一个安装Microsoft Visual C++ 2008的软件安装提示框，点击Next下一步即可。

  • 点击选中I have read and accept the license terms复选框以同意此协议。点击Install进行安装动作。

  • 安装完成之后点击Finish按钮结束安装，并跳转到Proteus安装界面。

  • 点击Next下一步。

  • 点击勾选I accept the terms of this agreement复选框，然后点击Next进入下一步。

  • 选择使用本地安装许可证书，点击Next进入下一步。

  • 在初次安装软件的时候需要安装许可证，点击下一步跳转到许可证安装界面。

  • 解压在软件包中的Licence.zip获取到Licence.lxk，点击安装界面上的Browse For Key File按钮，在选择文件框中选择Licence.lxk文件。

  • 获取到许可证书后点击Install按钮进行安装。

  • 安装成功后的界面如下，点击Close即可关闭。

  • 在此界面无需选择，点击Next进入下一步。

  • 在安装选项的界面选择Typical选项。

  • 等待安装成功，大致需要等待2分钟，不同的电脑性能，时间有所差异。

  • 安装成功之后的界面如下，此时不要点击Run Proteus 8 Professional，点击Close，进入破解步骤。
