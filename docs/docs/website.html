# 网站开发

为开发网站，前期需要您提供几项内容

# 1.域名

例如：winstonkenny.cn

域名可以在阿里云或者腾讯云购买，这两个平台购买域名可以申请免费的ssl证书

# 2.服务器

在购买域名的平台中购买至少一台服务器，如果你的网站同时访问的人不超过30人，可以购买最便宜的服务器

CPU：1核

内存：2G

硬盘：50G

网络带宽：1M
