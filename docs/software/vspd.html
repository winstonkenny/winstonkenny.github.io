# Virtual Serial Port Driver

Virtual Serial Port Driver是虚拟串口软件，主要的用处是在电脑上建立一个虚拟的串口通道，实现连个串口之间的通信，主要应用场景是调试计算机软件程序与实际的串口设备之间的数据连接，其中之最为典型的是上位机软件，上位机软件是为了监控设备的数据而存在的一种计算机软件，例如在电脑上监控下位机设备的数据信息，比如温湿度传感器的温湿度信息，烟感传感器的烟雾浓度数据，以及远程水位监测的水位数据等，都可以在上位机上以图表或者是图形的方式进行显示，有利于进行人为监控或者是非人为自动监控。虚拟串口则可以在计算机中模拟现实中的设备，对上位机发送相应的数据信息，以测试或者演示上位机功能，在研发的雏形阶段，虚拟串口工具能够得以广泛使用。

# 安装

  • 软件程序压缩包中包含以下内容，Virtual Serial Port Drive.exe 是安装程序包，破解汉化补丁是用来汉化软件的文件。

  • 双击打开Virtual Serial Port Drive.exe文件，显示如下安装界面，点击Next按钮进行下一步安装。

  • 选择I accept the agreement 单选框以同意此许可协议，点击Next按钮进行下一步。

  • 如下将软件安装到此路径下，此路径也是软件破解所需要的路径。一直点击Next下一步，直到软件安装完成。

# 破解

  • 软件安装成功后，将破解汉化补丁包中的文件复制到软件安装目录下。

  • 将目录下的同名文件替换掉，即可完成软件破解。
